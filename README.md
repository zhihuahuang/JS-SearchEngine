# JS-SearchEngine
A Search Engine JavaScript Code


## Browers Support
- Firefox 4+
- Chrome 29+
- Opera 16+
- Safari 8+

##Use Like
    window.SearchEngine.submit(keyword, engine[, type]);

Or

    window.SearchEngine.submit({
        keyword: 'string',
        engine: 'string',
        type: 'string'
    });
